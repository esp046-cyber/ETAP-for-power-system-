# ETAP-for-power-system
 mobile-optimized guide for KOCOS ACTAS C24 with ETAP integration. Features:
Key Elements:
Collapsible sections for efficient navigation
Comprehensive testing procedures (timing, travel, resistance, coil analysis)
ETAP workflow integration (export/import, relay coordination, arc flash)
Step-by-step instructions with visual hierarchy
Technical specifications and quick reference tables
Troubleshooting and maintenance protocols
1% margins, compact font sizing, no search
ETAP Focus:
CSV/XML export procedures
TCC curve generation from real test data
Short circuit and arc flash analysis integration
Relay coordination workflows
Protection study data mapping
All formatted for mobile viewing with gradient background, collapsible menus, and minimal spacing as requested.
